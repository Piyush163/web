4.Canvas and SVG tags
Ans.
Canvas

1.Raster based "composed of pixel"
2.Canvas has poor scalability. Hence it is not suitable for printing on higher resolution.
3.Canvas gives better performance with smaller surface or larger number of objects.
4.Canvas can be modified through script only.
5.Single element similar to in behavior. Canvas diagram can be saved to PNG or JPG format.

SVG

1.Vector based "composed of shapes"
2.SVG has better scalability. So it can be printed with high quality at any resolution.
3.SVG gives better performance with smaller number of objects or larger surface.
4.SVG can be modified through script and CSS.
5.Multiple graphical elements, which become the part of the page's DOM tree. HTML5 Canvas Tag
